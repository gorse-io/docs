# 控制台

主节点提供 Gorse 仪表盘，可通过主节点的 HTTP 端口（默认值为 `8088`）访问。HTTP 主机和 HTTP 端口在配置文件的 [`[master]`](./config#master) 部分中指定。

## 授权

默认配置允许未经授权访问控制台。

- **用户名和密码：** 仪表盘的用户名和密码可以在配置文件的 [`[master]`](./config#master) 部分中设置。如果设置了用户名和密码，访问仪表盘时将显示登录页面。

- **OpenID Connect：** 启用并配置配置文件的 [`[oidc]`](./config#oidc) 部分。如果启用了 OpenID Connect，访问仪表盘时，您将被重定向到身份提供商进行身份验证。

## 指南

仪表盘提供用于监控和管理推荐系统的各种功能。主要功能如下所述。

### 系统概览

在仪表盘的“概览”页面中，您可以查看系统的总体状态，包括：
- 用户、物品和反馈的数量。
- 协同过滤推荐器的 NDCG、精确率和召回率。
- 排序器（因子分解机）的 AUC、精确率和召回率。
- 非个性化推荐和最新物品。

### 任务状态

在仪表盘的“任务”页面中，您可以查看所有正在运行的任务和已完成任务的状态。信息包括任务名称、任务状态、开始时间、结束时间和进度。

### 集群状态

在仪表盘的“集群”页面中，您可以查看集群中所有节点的状态，包括服务器节点和工作节点。信息包括节点类型、主机名、UUID 和版本。

### 用户管理

在仪表盘的“用户”页面中，用户在表格中列出。您可以执行以下操作：

- 单击 ::view:: 按钮导航到用户详细信息页面。在用户详细信息页面中，您可以查看用户详细信息和来自用户到用户推荐器的用户邻居。

- 单击 ::likefill:: 按钮查看用户的反馈和推荐。在反馈和推荐页面中，您可以查看用户的反馈历史记录和来自各种推荐器的推荐物品。

- 单击 ::delete:: 按钮删除用户。

### 物品管理

在仪表盘的“物品”页面中，物品在表格中列出。您可以执行以下操作：

- 单击 ::view:: 按钮导航到物品详细信息页面。在物品详细信息页面中，您可以查看物品详细信息和来自物品到物品推荐器的物品邻居。

- 单击 ::delete:: 按钮删除物品。

### 高级操作

在仪表盘的高级页面中，您可以执行以下操作：

| 操作         | 说明                                                                         |
|--------------|------------------------------------------------------------------------------|
| 导出用户     | 单击“导出用户”按钮将所有用户数据导出为 JSON lines 文件。                     |
| 导出物品     | 单击“导出物品”按钮将所有物品数据导出为 JSON lines 文件。                     |
| 导出反馈     | 单击“导出反馈”按钮将所有反馈数据导出为 JSON lines 文件。                     |
| 导入用户     | 1. 单击“导入用户”按钮导航到“导入用户”页面。<br>2. 单击“浏览”选择包含用户数据的 JSON lines 文件。<br>3. 单击“确认导入”开始从所选文件导入用户。 |
| 导入物品     | 1. 单击“导入物品”按钮导航到“导入物品”页面。<br>2. 单击“浏览”选择包含物品数据的 JSON lines 文件。<br>3. 单击“确认导入”开始从所选文件导入物品。 |
| 导入反馈     | 1. 单击“导入反馈”按钮导航到“导入反馈”页面。<br>2. 单击“浏览”选择包含反馈数据的 JSON lines 文件。<br>3. 单击“确认导入”开始从所选文件导入反馈。 |
| 清除数据库   | 单击“清除数据库”按钮删除数据库中的所有数据。<br>此操作不可逆，请谨慎操作。   |

### 系统设置

在仪表盘的“设置”页面中，您可以查看系统设置。

::: warning
尚不支持通过仪表盘修改推荐配置。将在未来版本中支持。
:::
